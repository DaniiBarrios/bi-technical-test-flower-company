# Prueba Técnica BI – Análisis de Distribución de Flores

# Project Overview  
# Descripción del Proyecto

**English**

This repository contains the solution for a Business Intelligence technical assessment based on a flower distribution company scenario.  
The goal of the analysis is to evaluate regional sales performance, shipment efficiency, and inventory utilization using sample shipment and sales data.

**Español**

Este repositorio contiene la solución a una prueba técnica de Business Intelligence basada en un escenario de una empresa de distribución de flores.  
El objetivo del análisis es evaluar el desempeño de ventas por región, la eficiencia de los envíos y la utilización del inventario usando datos de ventas y despachos.
