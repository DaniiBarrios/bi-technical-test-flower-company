# Data Analysis  
# Análisis de Datos

**English**

The analysis focuses on understanding sales performance and inventory efficiency through the **sell-through rate**, defined as the proportion of shipped units that were successfully sold.

Sell-through = Units Sold / Units Shipped

This metric helps identify:

- high-performing regions
- potential overstock issues
- shipment allocation efficiency

**Español**

El análisis se centra en comprender el desempeño de ventas y la eficiencia del inventario mediante la métrica **sell-through**, definida como la proporción entre unidades enviadas y unidades vendidas.

Sell-through = Unidades Vendidas / Unidades Enviadas

Esta métrica permite identificar:

- regiones con mejor desempeño
- posibles problemas de sobreinventario
- eficiencia en la asignación de inventario

---

# Key Findings  
# Hallazgos Clave

**English**

- Arizona achieved the highest sell-through rate at **87%**, exceeding the 80% target.
- Texas showed moderate performance with **66.7% sell-through**.
- New Mexico had the weakest performance with **60%**, suggesting potential over-shipment relative to demand.

**Español**

- Arizona alcanzó el mayor sell-through con **87%**, superando el objetivo del 80%.
- Texas mostró un desempeño moderado con **66.7%**.
- New Mexico tuvo el desempeño más bajo con **60%**, lo que sugiere un posible exceso de inventario frente a la demanda.
